\chapter*{Abstract}
\label{chap:abstract}
\addcontentsline{toc}{chapter}{\nameref{chap:abstract}}

Prometheus is an integrated AI system designed to bridge the gap between gym infrastructure and the member experience by 
repurposing existing CCTV networks into real-time analytical engines. Traditional gym management relies on manual tracking 
or expensive hardware sensors, both of which fail to provide real-time, scalable data. This project implements a 
Computer Vision pipeline using the YOLO (You Only Look Once) architecture to detect equipment occupancy and analyze spatial 
traffic patterns. The system provides a dual-interface solution: an iOS application for members to check live machine availability 
and a web-based dashboard for owners to visualize facility utilization via heatmaps. By leveraging Edge Computing to ensure 
privacy and low latency, Prometheus optimizes workout efficiency for users and provides data-driven insights for facility 
operators to improve equipment Return on Investment (ROI).
